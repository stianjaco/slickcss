Using the Grid

Creating a grid with SlickCSS is as simple as it gets. You just need to add a row .row and your columns with .col
By default the grid-system supports up to 12 columns. By default each column takes up equal amount of space, you can spesfiy the space with a modifier, like .col-3

Example:

I'm a column

I'm a column

Code:
<div class="row">
	<div class="col">
		<p>I'm a column</p>
	</div>
	<div class="col-3">
		<p>I'm a column</p>
	</div>
</div>

You can also define width of each column based on the viewport by adding viewport modifier and size, like .col-md-6
